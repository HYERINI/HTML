1. 텍스트 굵게 강조 : strong으로 굵게 or b로 굵게
2. 텍스트 기울이기 : em으로 기울이기 or i로 기울이기
3. 줄임말 표시하고 title속성 함께 사용 -> abbr 태그 사용 IoT
4. 참고 내용 표시 -> cite 태그 : 이걸 인용했어~
5. 컴퓨터 인식 위한 소스코드 -> code태그 : source code
6. 텍스트 작게 쓰기 : small로 작게
7. 아래 첨자 -> sub 태그 : H2O
8. 위 첨자 -> sup 태그 : x2
9. 취소선 긋기 -> s 태그 : 짝대기가 그어진다
10. 공동작업문서에서 새로운 내용 삽입 : ins태그로 새로운 내용 삽입
11. 공동작업문서에서 기존 내용을 삭제 : del태그로 기존 내용 삭제
12. 순서있는 목록 : ol + li
13. 순서없는 목록 : ul + li
14. 설명 목록 : dl + dt + dd
