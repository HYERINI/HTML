[[ CSS 기초문법 ]]

선택자 (Selector) 
: 스타일을 적용하고자 하는 HTML 요소를 선택하는 역할

속성 (Property) 
: 지정할 스타일의 속성명에 해당
: (속성 : 값;) 이 한 단위
  ;(세미 콜론)을 이용하여 구분
  
값 (Value)
: 키워드나 특정 단위를 이용하여 원하는 스타일을 적용
: 속성(Property)와 쌍을 이룸

선언 블록 (Declaration block)
: {} 중괄호로 구분한다
